---README File for running OP Bank web app front and back end---

---Front-end---
1. Unzip and extract the folder 'op-bank-web-master_new' on your PC
2. Open the folder in VS Code
3. Open a new terminal and type the cmd 'npm install' and then 'npm run dev'
4. Open the specified URL in the browser

The app front-end is now accessible.

---Back-end----
1. Make sure to have MySql installed on your machine
2. Unzip and extract the folder 'op-bank-backend-sql-master' on your PC
3. Open the folder in VS Code
4. Open a new terminal and type the cmd 'npm install' and then 'npm run dev'
5. The server should now be running

To access the database, view the config.json file in a code editor. This containts the user and password details.
